# 🏪 烨权销售追踪看板

> 纯前端零售销售数据分析工具，支持多系统 Excel 数据上传，即开即用，无需后端。

---

## 💡 为什么做这个？

国内很多加盟系统自带的后台**没有数据分析功能**——看不出爆品是什么，不知道哪个门店该补货，只能靠感觉经营。作为超级加盟商，自己也深有体会：每次想知道"这周卖得最好的款是什么"，都要手动翻 Excel，费时费力还容易出错。

这个看板就是为了解决这个问题而做的。把系统导出的 Excel 丢进来，马上就能看到各店销售排名、爆款单品、库存预警，不需要任何技术背景，打开浏览器就能用。

**欢迎大家下载并自由修改，改成适合自己店铺或超级加盟商体系的专属看板。** 🙌
如果你也是经营同品牌的加盟商，数据格式完全兼容，可以直接下载使用，无需任何修改。

## ✨ 功能概览

| 模块 | 说明 |
|------|------|
| 🏬 **店铺追踪** | 左侧门店列表 + 右侧 Top 20 销售单品明细，支持颜色级别库存预警 |
| 🏷️ **品牌分析** | 按品牌查看 Top 30 爆款，展开查看各店铺销售 & 库存拆解 |
| 📊 **门店排名** | 全体门店成交额横向对比排名 |
| 🏆 **全网爆款** | 跨门店汇总热销款，支持按成交额 / 件数 / 在售天数排序 |
| 📅 **时间分析** | 按日期区间查看销售趋势 |
| 📦 **库存分析** | 库存健康度一览，识别售罄 / 低库存风险款 |

---

## 🗂️ 支持数据源

上传一个 Excel 文件（`.xlsx`），需包含以下两个工作表：

| 工作表名 | 系统来源 | 说明 |
|----------|----------|------|
| `销售` | 创美 / 拉夏 / 大码 | 销售流水，需含店仓、品牌、款号、品名、颜色、数量、金额、日期 |
| `库存` | 创美 / 拉夏 / 大码 | 当前库存快照，需含店仓、款号、品名、颜色、库存数量、金额 |

> 三套系统（创美、拉夏贝尔、大码女装）的字段结构已内置自动识别，无需手动转换。

## 🚀 快速使用

### 在线访问
直接打开 GitHub Pages 链接即可使用（零安装）。

### 本地运行
```bash
# 克隆仓库
git clone https://github.com/你的用户名/your-repo.git

# 直接用浏览器打开 index.html 即可
open index.html
```

### 数据上传流程
1. 打开页面，点击顶部上传区域，选择 Excel 文件
2. 数据在**浏览器本地**解析，不会上传至任何服务器
3. 使用日期筛选器设定分析周期
4. 切换各 Tab 查看不同维度报表

## 📊 核心指标说明

- **零售金额**：商品标牌价 × 销售件数（未打折价格）
- **成交金额**：实际收款金额（折扣后）
- **折扣率**：成交金额 / 零售金额，用于识别高折扣款
- **库存预警**：
  - ⛔ **售罄**：有销售记录但当前库存 = 0
  - ⚠️ **低库存**：当前库存 ≤ 2 件

```javascript
// 品牌颜色点映射
const STORE_DOT = {
  "创美": "#6c63ff",
  "江苏": "#4ecdc4",
  // 在此添加新品牌...
};
```

## 📝 注意事项

- 所有数据处理均在**浏览器本地**完成，刷新页面后需重新上传文件
- 推荐使用 Chrome / Safari 最新版本以获得最佳性能
- 大文件（> 5000 行）建议按月拆分上传，避免渲染卡顿
